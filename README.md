# Profit Share Calculator

This is a lightweight browser-based profit share calculator.

## Current behaviour

- Calculates `Corporate Factor = (ROIC - 5%) + (2 x Organic Growth) + Acquired Growth`
- Treats the Corporate Factor result as a percentage and applies it as a decimal in `Bonus = Corporate Factor x PF x Base Salary`
- Saves `PF`, `Base Salary`, and the Corporate Factor inputs in browser local storage

## Open the calculator

Open `index.html` in a browser.
